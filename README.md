# Sirsak Pop Nasabah

Turn your waste into rewards while making a real environmental impact.

## About

Sirsak Pop Nasabah is a mobile app that connects customers to Indonesia's largest traceable waste value chain network. Find nearby collection points, drop off recyclable waste, earn Sirsak Points, and redeem rewards - all while tracking your environmental contribution.

## Features

- **Find Collection Points** - Discover nearby Bank Sampah and RVM locations with GPS-powered search
- **Earn Rewards** - Get Sirsak Points for every waste deposit
- **Dual Wallet** - Track Sirsak Points and Bank Sampah balance separately
- **Environmental Impact** - Monitor your waste collected, recycled, and carbon emissions avoided
- **Redeem** - Exchange points for cash, vouchers, or exclusive products
- **Bilingual** - Available in Bahasa Indonesia and English

## How It Works

1. **Search** - Find your nearest waste collection points
2. **Drop** - Dispose your separated recyclable waste
3. **Earn** - Get Sirsak Points for every transaction
4. **Redeem** - Exchange points for rewards

## Tech Stack

- Flutter 3.29.0
- Dart 3.7.0
- Riverpod (State Management)
- GoRouter (Navigation)
- Freezed (Immutable State)

## Getting Started

```bash
# Install dependencies
flutter pub get

# Generate code (Freezed, JSON serializable)
flutter pub run build_runner build --delete-conflicting-outputs

# Run the app
flutter run
```
